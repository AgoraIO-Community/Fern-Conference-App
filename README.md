# fern.

This project integrates Agora.io api to create a flexible Conferencing Application.

It supports: 
1. QUALITY TRANSPARENCY
2. SUPPORT FOR IPV6
3. DEVICE MANAGEMENT
6. VOLUME INDICATOR OF SPEAKING PARTICIPANT
